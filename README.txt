-Unzip the program files:
    1. From the command line: $ unzip hw2.zip

-In order to run the proof of function program, merge4.py:
    1. Make sure the program files are located in a folder with the data.txt file containing integers to be sorted.
    2. To run merge4 sort: $ ./merge4.py
    3. Result file named merge4.txt, containing the sorted integers, will be created in the current folder.

-In order to run the timing program merge4Time.py:
    1. In the folder containing the program file, run: $ ./merge4Time.py
    2. Timing results will be displayed as terminal output.
